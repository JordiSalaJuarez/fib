Para ejecutar un ejemplo, simplemente ejecuta desde un terminal el comando

  clips -f ejemplo_vegetariano_moderno.bat

Si quieres personalizar el input que recibe el programa, escribe en la 
terminal

  clips -f load.bat

Y escibe (run) una vez te encuentres en la terminal de clips

En windows tambien puedes hacerlo si cargas los .bat con la opcion 

  Load Batch...

Espero disfrutes con el programa tanto como nosotros programandolo :)
